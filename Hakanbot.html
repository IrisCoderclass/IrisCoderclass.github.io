Warning, the following is a very unrealistic story about our informatics teacher in the form of a bunch of "news articles".

Legendary coding language created.

A few days ago coder Hakan Akkas discovered the LScript(Legendary Script) this coding language is capable of doing anything from cloning someone's intelligence to creating food out of thin air. Using this coding language Hakan is planning on cloning his own intelligence into a bot that will obey him. The LScript is very hard to learn and Hakan is currently the only one who is capable of using it.

The Omnicrete.

Hakan Akkas has created a machine called the Omnicrete this machine is capable of creating anything, form materials to food to computers. Because this machine could potentially be dangerous if it fell into the wrong hands Hakan is planning on hiding it somewhere safe.

Hakanbot-0 created.

The first Hakanbot has been created using the LScript, this was supposed to be a robot with the intelligence and mind of it's creator Hakan Akkas, however it isn't as smart as it should have been and isn't capable of human speech. It does obey Hakan Akkas, who is now planning on using it to protect the omnicrete.
